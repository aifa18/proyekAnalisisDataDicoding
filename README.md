# E-Commerce Data Analysis Dashboard 📊

## Struktur Proyek
- `dashboard/`: Berisi file utama dashboard (`app.py`) dan data yang sudah dibersihkan (`main_data.csv`).
- `data/`: Berisi file CSV mentah (sumber data asli).
- `notebook.ipynb`: Dokumentasi lengkap proses analisis dari Gathering hingga Conclusion.
- `requirements.txt`: Daftar library Python yang dibutuhkan.

## Cara Menjalankan Dashboard
1. Pastikan Anda memiliki Python terinstal.
2. Install semua library yang dibutuhkan:
   ```bash
   pip install -r requirements.txt
